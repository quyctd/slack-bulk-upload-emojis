# Slack - BULK Upload Emojis
---

## Install
1. Clone repo
```bash
git clone git@github.com:quyctd/slack-bulk-upload-emojis.git
```
2. Install dependencies
```bash
cd slack-bulk-upload-emojis
bundle install
```
## Usage
Just run this file
```bash
ruby slack_bulk_uploader.rb
```
